# Security Policy

## Supported versions

| Version | Supported |
|---------|----------|
| 1.x     | ✅ Yes    |

## Reporting a vulnerability

If you find a security issue in this tool itself (not in targets you scan with it), please report it responsibly.

**Do not open a public issue.**

Contact: [info@theoffsecgirl.com](mailto:info@theoffsecgirl.com)  
Include: description, reproduction steps, potential impact.

Expected response time: **72 hours**.

## Scope

This tool is intended for authorized security testing only.  
Any misuse is outside the scope of this project and the sole responsibility of the user.
